Базовые функции (входные елемент "directory"):
	б1) количество уроков по предмету в день ( дата, предмет, класс ) -> число/пустое множество
	б2) запись в журнале по предмету на дату по ученику по номеру урока ( ученик, дата, предмет, класс, порядковый номер урока ) -> список строк/пустое множество
	б3) оценка промежуточной аттестации по ученику, по предмету по виду ( ученик, предмет, вид аттестации ) -> число/пустое множество
	б3-1) результаты промежуточной аттестации по ученику по предмету ( таблица, ученик ) ->массив/пустое множество
	
Производные - 1:
	п1) отметки по ученику и предмету за период ( ученик, предмет, период )
	п2) только оценки по ученику и предмету за период ( ученик, предмет, период )
	п3) количество пропусков за период по ученику и по предмету ( ученик, предмет, период )
	
Производные - 2:
	пп1) оценки промежуточной аттестации класса по предмету за период ( класс, предмет, период )
	
Расчетные:
	р1) количество оценок по видам по ученику, предмету за период ( п2/б3 )
	р2) качество образования( п2/б3 )
	р3) успеваемость ( п2/б3 )
